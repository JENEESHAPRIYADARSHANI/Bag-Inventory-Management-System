========================================
   WHAT DOCKER DOES
========================================

Before Docker:
  - Start MySQL manually
  - Run: mvn spring-boot:run
  - Run: npm run dev
  - Manage 3 terminals

With Docker:
  - Run: docker-compose up
  - Everything starts automatically!

========================================
   SERVICES
========================================

MySQL Database:
  - Port: 3306
  - Database: quotation_db
  - Data persisted in Docker volume

Backend (Spring Boot):
  - Port: 8080
  - API: http://localhost:8080/api
  - Auto-connects to MySQL

Frontend (React):
  - Port: 5173
  - URL: http://localhost:5173
  - Served by Nginx

========================================
   COMMON COMMANDS
========================================

Start services:
  docker-compose up --build

Start in background:
  docker-compose up -d

Stop services:
  docker-compose down

View logs:
  docker-compose logs

Check status:
  docker ps
